# **App Name**: Planning Event

## Core Features:

- Client Event Request: Clients can create detailed event requests by specifying date, location, event type, budget, and desired professional categories. These requests are then published to relevant professionals.
- Professional Demand Dashboard: Professionals receive and view a filtered list of new client event requests relevant to their services and geographic area, enabling quick response actions.
- Secure Communication & Quote Management: A centralized messaging system for direct client-professional communication, allowing professionals to send detailed quotes and clients to review, compare, discuss, and electronically sign agreements.
- Professional Calendar & Availability: Professionals can manage their service calendar, mark specific dates as unavailable, and view all confirmed events to organize their schedule efficiently.
- AI-Powered Response Assistant: An AI tool that assists professionals in drafting personalized initial responses and suggests relevant quote template details based on the client's event request and their service offerings.
- User Profile & History: Both clients and professionals can manage their profiles, view a history of their past requests/services, and access signed contracts and invoices.

## Style Guidelines:

- The chosen light color scheme focuses on clarity and connection, ideal for a professional planning tool. The primary color is a deep, inviting blue, #2456B3. The background color is a very light, desaturated blue, #ECF1F9, ensuring high readability. The accent color, a vibrant teal, #26CBED, adds visual interest and highlights important interactive elements, positioned analogously to the primary on the color wheel.
- For headlines, 'Space Grotesk' (sans-serif) will provide a modern, organized, and slightly technical feel. For body text and longer content like descriptions and contracts, 'Inter' (sans-serif) will ensure excellent readability and a neutral, professional appearance.
- Utilize a clean, modern set of outline icons. Icons should clearly represent event categories, planning actions, communication, and contract management (e.g., calendar, message bubble, document icon, checkmark).
- Emphasize a structured and organized layout with generous white space. Use grid-based designs to present complex information clearly, especially for professional dashboards and client request forms. Prioritize logical information hierarchy and intuitive navigation for both user types.
- Subtle and functional animations should be used to provide user feedback, such as smooth transitions for filtering results, clear visual cues upon form submission, or a gentle fade-in for newly loaded content sections.
